Implementación¶

Datos¶

El inters recae en comparar cuatro aplicaciones. Por simplicidad, se denotan las cuatro aplicaciones como tratamiento A, B, C y D. Se cuenta con 400 datos.

Variables

  • $y_i := \text{ Medida de de la aplicacin en el individuo } i$
  • $x_{i,1} := \text{ Variable de inters 1 para el individuo } i$
  • $x_{i,2} := \text{ Variable de inters 2 para el individuo } i$
  • $x_{i,j} =\begin{cases} 1, & \text{ si al individuo } i \text{ se le aplic } j, j=\left \{ A,B,C,D \right \} \\ 0 & \text{ e.o.c } . \end{cases}$

ANCOVA¶
Modelo de RLM¶
El modelo de RLM con interacciones esta parametrizado de la siguiente manera


$y_i = \beta_0 + \underbrace{\beta_1 x_{i,1}}_{Variable 1} + \underbrace{\beta_2 x_{i,2}}_{Variable 2} + \underbrace{\beta_3 x_{i,B} + \beta_4 x_{i,C} + \beta_5 x_{i,D}}_{Tratamiento} + \underbrace{\beta_6 x_{i,B}x_{i,1} + \beta_7 x_{i,C}x_{i,1} + \beta_8 x_{i,D}x_{i,1}}_{\text{Tratamiento con variable 1}} + \underbrace{\beta_9 x_{i,B}x_{i,2} + \beta_{10} x_{i,C}x_{i,2} + \beta_{11} x_{i,D}x_{i,2}}_{\text{Tratamiento con variable 2}} + \epsilon_i$


donde $\epsilon_i \text{N}\left ( 0, \sigma^2 \right ), \forall i\in \left \{ 1 , \dots , n  \right \}. $


Observacin: En la parametrizacin del modelo se consider la aplicacin $A$ como el nivel de referencia, entonces


    

  • Si al individuo $i$ se aplica $A$, entonces $x_{i,B}=x_{i,C}=x_{i,D}=0$
    • 


$$
\hat{\mathbb{E}}\left [ y_i; X A , x_i \right ] = \hat{ \beta}_0 + \hat{\beta}_1x_{i,1} + \hat{\beta}_2x_{i,2}
$$
    

  • Si al individuo $i$ se aplica $B$, entonces $x_{i,B}=1$ y $x_{i,C}=x_{i,D} = 0$
    • 


$$
\hat{\mathbb{E}}\left [ y_i; XB , x_i \right ] = \left ( \hat{ \beta}_0 + \hat{\beta}_3 \right ) + \left ( \hat{ \beta}_1 + \hat{\beta}_6 \right )x_{i,1} + \left ( \hat{ \beta}_2 + \hat{\beta}_9 \right )x_{i,2}
$$
    

  • Si al individuo $i$ se aplica $C$, entonces $x_{i,C}=1$ y $x_{i,B}=x_{i,D} = 0$
    • 


$$
\hat{\mathbb{E}}\left [ y_i; XC , x_i \right ] = \left ( \hat{ \beta}_0 + \hat{\beta}_4 \right ) + \left ( \hat{ \beta}_1 + \hat{\beta}_7 \right )x_{i,1} + \left ( \hat{ \beta}_2 + \hat{\beta}_{10} \right )x_{i,2}
$$
    

  • Si al individuo $i$ se aplica $D$, entonces $x_{i,D}=1$ y $x_{i,B}=x_{i,C} = 0$
    • 


$$
\hat{\mathbb{E}}\left [ y_i; XD , x_i \right ] = \left ( \hat{ \beta}_0 + \hat{\beta}_5 \right ) + \left ( \hat{ \beta}_1 + \hat{\beta}_8 \right )x_{i,1} + \left ( \hat{ \beta}_2 + \hat{\beta}_{11} \right )x_{i,2}
$$

Reducción del modelo¶
Para poder reducir el modelo, es necesario verificar


    

  1. El modelo tiene sentido? (interpretar la prueba asociada a la tabla ANOVA). Se plantea la prueba de hiptesis:
    2. 



$H_0: \beta_1 = \beta_2 = \cdots = \beta_{11} = 0 \text{ v.s. } H_a = \beta_i \neq 0, \text{ para alguna } i = 1, \dots , 11 $











Tiene sentido pues el p-value de la prueba asociada a la tabla ANOVA es de: 5.156134400841888e-125

















    
    






























Por lo tanto, rechazamos la hipotesis nula. Concluimos que: Con una confianza del 95% las variables de inters (x1, x2) o alguna de las aplicaciones (A, B, C,D) nos ayudan a modelar el valor promedio de la condicicin.











    

  1. El efecto de las variables de inters (x1, x2) es el mismo en las 4 aplicaciones? (rectas con la misma pendiente). Se plantea la prueba de hiptesis: 
    2. 



$H_0: \beta_j= 0\text{ para toda } j \in\{6,\dots,11\}\text{ v.s. } H_a: \beta_j \neq 0 \text{ para algn }  j \in\{6,\dots,11\}$.

Normalidad¶
















    
    


















    
    

































    
    

































    
    

















    
    



























Los test Shapiro-Wilk test y D'Agostino's K-squared test son dos de los test de hiptesis ms empleados para analizar la normalidad. En ambos, se considera como hiptesis nula que los datos proceden de una distribucin normal.


El p-value de estos test indica la probabilidad de obtener unos datos como los observados si realmente procediesen de una poblacin con una distribucin normal con la misma media y desviacin que estos. Por lo tanto, si el p-value es menor que un determinado valor (tpicamente 0.05), entonces se considera que hay evidencias suficientes para rechazar la normalidad.

















    
    






Preuba de Shapiro-Wilk: Estadstico = 0.8112004995346069, p-value = 2.7095571219558445e-21


























    
    






Prueba de D'Agostino: Estadstico = 172.4237801934205, p-value = 3.619527299502789e-38





















    Por lo tanto, los residuales no tienen distribucin normal











Homocedasticidad¶
La suposicin de homoscedasticidad es una suposicin vital. Si esta suposicin es violada, entonces los errores estndar sern sesgados. Los errores estndar se utilizan para realizar pruebas de significacin y calcular los intervalos de confianza.


Esto puede ser probado usando un residual vs. valores ajustados, mirando un grfico de dispersin (si una forma de como est presente entonces heteroscedasticidad est presente), o mediante el uso de una prueba estadstica como Bruesch-Pagan, prueba de Cook-Weisberg, o prueba general blanca. En este ejemplo, se utilizar la prueba Bruesch-Pagan, la cul tienen como hiptesis nula que hay presencia de homocedasticidad.
